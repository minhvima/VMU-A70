# VMU A70

Ban web mobile cho mini game ky niem VMU 70 nam.

## Cau truc

- `docs/`: ban web san sang dua len GitHub Pages
- `mobile_web/`: thu muc lam viec/local source hien tai
- `serve_mobile_web.bat`: chay local server de test tren may cua ban

## GitHub Pages

De phat hanh cho nguoi dung cong khai:

1. Push repo len GitHub.
2. Vao `Settings > Pages`.
3. Chon `Deploy from a branch`.
4. Chon branch `main` va folder `/docs`.

Sau do website se co link public de mo tren dien thoai ma khong can cung Wi-Fi voi may tinh cua ban.

## Ghi chu

- Ban web da co logo, credit va gameplay chinh.
- Folder `docs/` la ban de deploy. Khi cap nhat giao dien, nho dong bo lai tu `mobile_web/`.
